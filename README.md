# frontend-prep

Track your progress through [BFE.dev](https://bigfrontend.dev/problem) JavaScript problems.

## How to use

- `✓` = Solved (with tests passing)
- `○` = Attempted / In progress
- `-` = Not started
- **Last revision** = from git history
- **Next revision** = when to revisit (spaced repetition: easy +2 weeks, medium +1 week)

## Commands

```bash
npm install
npm test                    # Run all tests
npm run test:watch          # Watch mode
npm run test:problem 001    # Run tests for problem 001
```
